# Aequitas--The-Ethical-Hiring-Auditor
Aequitas: An Ethical AI Auditor for recruitment. I built a Logistic Regression model achieving 94.67% F1-score while auditing systemic bias. Using VIF and AIR metrics, I mitigated historical bias, improving equity from 0.62 to 0.707. This project highlights the "Fairness Ceiling" and the need for high-integrity data in AI.
